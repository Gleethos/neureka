Autograd Tensor Integration Tests

This specification contains tests which cover the autograd behavior of tensors.
The classes involved in governing the tested features are the Tsr, GraphNode and Function (& implementations) classes.

Features:

  • Test pending error optimization
  • Test JIT propagation variant one.
  • Test JIT propagation variant two.
  • Gradient auto-apply kicks in when used AD uses JIT prop
  • Test no preemptive gradient apply when not requested and auto apply and JIT_prop
  • Test autograd without JIT and auto apply.
  • Test in-differential and JIT with auto apply
  • Test no JIT prop when forward AD
