Tensors Exception Behavior


This specification covers the behavior of tensors in exceptional situations.

Features:

  • Trying to inject an empty tensor into another causes fitting exception.
  • Passing an invalid object into Tsr constructor causes descriptive exception.
  • Passing an invalid key object into the "getAt" method causes a descriptive exception.
  • Out of dimension bound causes descriptive exception!
