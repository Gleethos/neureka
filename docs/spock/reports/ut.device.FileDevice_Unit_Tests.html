FileDevice Behavior


This specification covers the behavior of the "FileDevice" class, which enables the persistence of tensor data.

Features:

  • A file device stores tensors in idx files by default.
  • A file device stores tensors in various file formats.
A file device stores tensors in idx files by default. Return

A new tensor is being created for testing.
 
Tsr a = new Tsr([2, 4], [ 5, 4, -7, 3, -2, 6, -4, 3 ])
