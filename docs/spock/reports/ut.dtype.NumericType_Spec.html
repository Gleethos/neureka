The NumericType and its implementations model their respective numeric data types.
This specification covers the behavior of the NumericType interface
    which is responsible for modelling numeric data types which may or may not be native to the JVM. 
    These implementations however do not model them in the traditional OO style
    but merely expose useful utility method for converting and representing 
    these numeric data types using JVM types.
This specification covers implementations of the "interface neureka.dtype.NumericType" interface. Such classes are responsible for representing all numeric types including the ones which are foreign to the JVM, namely : unsigned integer types.

Features:

  • NumericType implementations return their expected properties.
  • NumericType implementations behave as expected.
  • Conversion goes both ways and produces expected numeric values.
  • NumericType conversion to holder types yields expected results.
