Framing Behavior


This specification covers the behavior of the classes contained in the "framing" package, which contains logic in order to provide the possibility to alias tensor indices.

Features:

  • Added labels to tensors are accessible through the "index()" method.
  • Rank 2 tensors can be labeled and their labels can be used to extract slices / subsets of tensors.
  • Rank 3 tensors can be labeled and their labels can be used to extract slices / subsets of tensors.
