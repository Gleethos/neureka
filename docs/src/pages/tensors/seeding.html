Stringed seeding:

Pseudo randomness can be achieved by using seeds. Strings are a good source of randomness when hashed, which is what Neureka does internally to produce new random numbers distributed around 0. When a seed is being passed to the constructor of a tensor it will serve as a source of randomness used to fill this new tensor instance with values. Equal strings will produce the same values. 



        def t1 = Tsr.of([2, 3], "I am a seed! :)")
        def t2 = Tsr.of([2, 3], "I am a seed! :)")
        def t3 = Tsr.of([2, 3], "I am also a seed! But different. :)")

        // t1.toString() == t2.toString()
        // t3.toString() != t2.toString()

                


        Tsr<Double> t1 = Tsr.of(Double.class, new int[]{2, 3}, "I am a seed! :)");
        Tsr<Double> t2 = Tsr.of(Double.class, new int[]{2, 3}, "I am a seed! :)");
        Tsr<Double> t3 = Tsr.of(Double.class, new int[]{2, 3}, "I am also a seed! But different. :)");

        // t1.toString() == t2.toString()
        // t3.toString() != t2.toString()

                


        val t1 = Tsr.of(Double::class.java, intArrayOf(2, 3), "I am a seed! :)")
        val t2 = Tsr.of(Double::class.java, intArrayOf(2, 3), "I am a seed! :)")
        val t3 = Tsr.of(Double::class.java, intArrayOf(2, 3), "I am also a seed! But different. :)")

        // t1.toString() == t2.toString()
        // t3.toString() != t2.toString()

Default seeding:

If you simply want to create random tensors without having to think about what kind of seed you want to supply, then simply resort to the following API. 



        var t1 = Tsr.ofRandom(Double, 2, 3)
        var t2 = Tsr.ofRandom(Double, 2, 3)

        // t1.toString() == t2.toString()

                


        Tsr<Double> t1 = Tsr.ofRandom(Double.class, 2, 3);
        Tsr<Double> t2 = Tsr.ofRandom(Double.class, 2, 3);

        // t1.toString() == t2.toString()

                


        val t1 = Tsr.ofRandom(Double::class.java, 2, 3)
        val t2 = Tsr.ofRandom(Double::class.java, 2, 3)

        // t1.toString() == t2.toString()
