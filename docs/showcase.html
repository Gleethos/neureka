Introduction

Neureka implements so called Auto-Differentiation. AD-algorithms are the backbone of today's most advanced artificial neural networks. It allows for calculating derivatives with respect to weight variables used to correct errors incrementally throughout many training iterations whereby neural networks then gain the ability to approximate, generalize and thereby learning. Differentiation goes both ways: Forwards and Backwards. Namely:
Forward-mode-AD and backward-mode-AD.
Neureka utilizes both techniques to get the best performance. Especially the use of backward-mode-AD revolutionized machine learning in recent decades. Gradient decent using backward-mode-ad is termed back-propagation and for most cases this approach accelerates gradient descent by orders of magnitudes compared to other techniques, making the difference between a model taking weeks instead of years to train. Beyond its use in machine learning, Auto-differentiation is a powerful computational tool in a multitude of other areas, ranging from weather forecasting to analyzing numerical stability. Besides it's use in machine learning, AD is a common tool for many other fields dependent on algebraic numerical computation. 


    Tsr x = new Tsr(3).setRqsGradient(true);
    Tsr b = new Tsr(-4);
    Tsr w = new Tsr(2);
    Tsr y = new Tsr(new Tsr[]{x, b, w}, "((i0+i1)*i2)^2");
    /* *
     *   f(x) = ((x-4)*2)^2; :=>  f(3) = 4
     *   f(x)' = 8*x - 32 ;  :=>  f(3)' = -8
     *
     *   y.toString(): "[1]:(4.0); ->d[1]:(-8.0), "
     * */

In this example Neureka calculated the derivative of y w.r.t x. Forward-AD is used due to the fact that no other dependencies need to be considered. By utilizying forward-AD Neureka greatly decreases Memory consumption as well as increase performance.
