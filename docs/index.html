But Java is verbose!

Popular deep learning libraries use python due to it's elegant Syntax. Java on the other Hand is considered verbose and bulky, so why would anybody ever write a deep learning library for Java? The answer is simple:
Nobody did!
This Java is written for every JVM-language, namely: Groovy, Kotlin, Scala, and Jython just to name a few. This library can be integrated by any language which compiles to JVM bytecode! Take a look at some groovy: 


        Tsr b = new Tsr([1], -4)
        Tsr w = new Tsr([1], 2)
        Tsr x = new Tsr([1], 3)

        Tsr y = b + w * x
        //  2 =-4 +(3 * 2)
                                                

            Tsr x = new Tsr(3).setRqsGradient(true);
            Tsr b = new Tsr(-4);
            Tsr w = new Tsr(2);
            Tsr y = new Tsr(new Tsr[]{x, b, w}, "((i0+i1)*i2)^2");
            /**
             *   f(x) = ((x-4)*2)^2; :=>  f(3) = 4
             *   f(x)' = 8*x - 32 ;  :=>  f(3)' = -8
             *
             *   y.toString(): "[1]:(4.0); ->d[1]:(-8.0), "
             * */
